📖 什么是 YOLO26？

YOLO（You Only Look Once）是计算机视觉领域最经典的实时目标检测算法系列。YOLO26 是 Ultralytics 最新发布的版本，将目标检测、实例分割、语义分割、图像分类、姿态估计、旋转框检测和目标跟踪统一到了一个框架中。


核心特性：

• 极速推理 — Nano 模型在 T4 GPU 上仅需 1.7ms，适合移动端和边缘设备

• SOTA 精度 — X-Large 模型在 COCO 上达到 57.5 mAP

• 六合一框架 — 检测、分割、姿态、分类、OBB、跟踪统一 API

• 20+ 导出格式 — ONNX、TensorRT、CoreML、OpenVINO、TFLite 等

• AGPL-3.0 开源 — 研究者和学生免费使用


模型体系（纯净态概念）：

YOLO26 的预训练权重就是「纯净态」——不可变的基准状态。在此基础上进行训练得到「训练态」，导出为部署格式得到「导出态」。完整谱系可追溯，SHA256 校验保证完整性。


版本信息：ultralytics v8.4.102 · PyTorch 2.13.0 · Python 3.13 · Apple MPS 加速
